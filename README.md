# 🏋️ Gym Management System (Full Stack)

A full-stack web application built using **Java Servlets (Jetty)** for the backend and **Vite + React (JavaScript)** for the frontend. This project allows users to register for gym services and stores data in a MySQL database.

---

## 🚀 Features

* 📝 User Registration Form
* 🔗 Frontend ↔ Backend API Integration
* 🗄️ MySQL Database Storage
* 🌐 CORS Handling
* ⚡ Fast UI with Vite + React

---

## 🛠️ Tech Stack

### 🔹 Frontend

* React (Vite)
* HTML, CSS, JavaScript

### 🔹 Backend

* Java Servlets
* Jetty Server

### 🔹 Database

* MySQL

## ▶️ How to Run

### 1️⃣ Start Backend

* Run Jetty server
* Backend runs on:

```
http://localhost:8080
```

### 2️⃣ Start Frontend

```
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```

---

## ⚙️ Configuration

* Update database credentials in:

```
DBConnection.java
```

* Ensure MySQL is running on port `3306`
